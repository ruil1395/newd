# Dota 2 Telegram Bot

Бот-аналитик для Dota 2 с живыми данными OpenDota. Работает на патче 7.40c.

## Возможности

- `/predict` — предсказание следующих пиков врагов и рекомендация своего пика на основе реальной статистики матчапов и меты.
- `/match <ID>` — детальный анализ матча по ID (урон, GPM, победитель, ссылка на Dotabuff).
- `/player <steam_id>` — анализ игрока: последние матчи, винрейт, любимые герои, средний KDA, GPM, XPM.
- `/hero <имя>` — текущий винрейт и пикрейт героя (данные OpenDota, обновляются каждый час).
